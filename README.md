# Grow for Jira

**Turn Jira work into real-world nature impact.**
By [afforestation.org](https://afforestation.org)

## Overview
Grow for Jira gamifies your team's workflow by converting completed issues into real trees. Configure scoring rules, funding portfolios, and visualize your impact directly within Jira.

## Features
- **Scoring Engine**: Award leaves based on Story Points, Status Categories, and more.
- **Impact Dashboard**: Visualize trees planted and carbon offset.
- **Funding Portfolio**: Choose which reforestation projects to support.
- **Privacy Controls**: Team-only or Org-wide leaderboards.

## Setup
1. Install the app in your Jira Cloud instance.
2. Go to **Apps > Afforestation Settings**.
3. Configure your Completion Rules and Funding Allocation.
4. Start closing issues and growing forests!
